# Python AI Agent (CLI Version)

A simple Python-based AI agent that interacts via text and performs multiple tasks using keyword-based NLP.

## Features
- Basic arithmetic calculations
- Wikipedia search and summary
- To-do list management
- Tells jokes

## How to Run

1. Clone the repository
2. Install dependencies:

## pip install -r requirements.txt

3. Run the agent:


## python main.py

## Example Commands
- calculate 10 + 5
- wiki Artificial Intelligence
- add todo buy groceries
- show todo
- tell me a joke
- exit
